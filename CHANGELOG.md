# Changelog

## [0.1.0] - 2026-04-02

### Initial Release / 初回リリース

- NMOS IS-05 signal routing control from Stream Deck  
  Stream DeckからNMOS IS-05信号ルーティングを操作
- Receiver selector buttons with visual selection state  
  選択状態をビジュアルで確認できるレシーバーセレクターボタン
- Sender (TAKE) buttons with BCC node integration or manual IS-04 URL  
  BCC連携または手動IS-04 URL入力に対応したセンダー（TAKE）ボタン
- ST 2110-7 redundant stream support (primary/secondary legs)  
  ST 2110-7 冗長ストリーム対応（プライマリ／セカンダリ）
- Compatible with any NMOS IS-04/IS-05 compliant device  
  NMOS IS-04/IS-05準拠デバイスと幅広く互換
- Works standalone or alongside Browser-Based Controller (BCC)  
  BCC（ブラウザベースコントローラー）との併用・単独運用の両方に対応
